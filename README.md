# Gerenciamento de Configuração

Material da aula sobre **Gerenciamento de Configuração**, abordando conceitos de:

- Controle de versões;
- Gerenciamento de mudanças;
- Gerenciamento de releases;
- Baseline e codeline;
- Sistemas centralizados e distribuídos;
- Git e GitHub;
- Branches, commits e merge.

---

# Objetivos da Aula

Ao final desta aula, o aluno deverá ser capaz de:

- Compreender os conceitos fundamentais de gerenciamento de configuração;
- Entender os processos de gerenciamento de mudanças e controle de versões;
- Identificar a importância do versionamento no desenvolvimento colaborativo;
- Aplicar conceitos de branch, commit, checkout e merge utilizando Git;
- Relacionar práticas modernas de controle de configuração com Engenharia de Software e DevOps.

---

# Conteúdo

## Gerenciamento de Configuração
- Conceitos fundamentais;
- Controle e rastreamento de alterações;
- Baseline e codeline.

## Gerenciamento de Mudanças
- Change Request (CR);
- Fluxo de mudanças;
- Avaliação de impacto;
- Rastreabilidade.

## Gerenciamento de Versões
- Controle de versões;
- CVS, SVN e Git;
- Sistemas centralizados e distribuídos.

## Gerenciamento de Releases
- Construção do sistema;
- Versionamento;
- Releases e distribuição.

## Prática com Git
- git init
- git clone
- git checkout
- git branch
- git commit
- git merge
- git push
